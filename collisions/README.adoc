= Ball Collision

Creates random balls. These balls will resolve any collision issues. Launching a
ball will show you an example of elastic collision.

checkout this link https://raw.githack.com/A-Larsen/balls/refs/heads/collisions/docs/formulas.html[formulas].
for a list of forumlas used.

== Controls

[%header,cols="1,1"]
|===
| control                             | action
| right-click                         | select ball
| right-click and left-click and drag | flings ball on release, the distance
                                        away from the ball determines the amount
                                        of velocity to launch the ball at
|===
